apm 更换为国内源

2020-02-25 #atom

要设置apm使用的软件源很简单，执行下列命令：
🙋‍♂

apm config set registry npm_mirror_url

将上面的npm_mirror_url替换为你想要使用的镜像源。如要使用淘宝源，即可以使用以下命令。

apm config set registry http://registry.npm.taobao.org

apm install --check

检查是否更换成功，出现

checking foring native build tools done
